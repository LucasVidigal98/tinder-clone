## Test in your computer

```sh
git clone https://github.com/LucasVidigal98/tinder-clone.git
```

```sh
cd tinder-clone
```

```sh
npm install && npm start
```
## Firebase

Put your Configuration on firebase.js and:

```sh
npm install firebase-tools -g
```

```sh
firebase init
```

```sh
npm run build
```

```sh
firebase deploy
```
